# **App Name**: TrustPilot Recommender

## Core Features:

- Use-Case Questionnaire: Admin-defined questionnaire for smartphones, earphones/headphones and laptops that record user's answers for use-case matching.
- Recommendation Engine: Analyze user's questionnaire answers against product data to generate a rank-ordered recommendation of the top 3-5 products. Session token must be used.
- Honest Product Details: Display product details with clearly stated 'Good for', 'Not good for', and 'Common regret reasons'.
- Real-World Scores: Show comfort score, reliability score, repurchase intent, and longevity estimates for each product.
- Scenario-Tagged Review System: Allow anonymous users to submit scenario-tagged reviews with comfort ratings, and textual reviews with upvotes.
- Ranking Explanation Tool: A tool to display a clear, detailed breakdown of why each product ranked the way it did. Breakdown includes: Use-case match, Real-world experience, Longevity, Budget alignment, and Buy-timing advisor.
- AI Review Moderation: Flag reviews with a high likelihood of being fake for admin moderation, reducing overall moderation overhead

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke calmness and trust.
- Background color: Light gray (#F0F0F0) for a clean, minimal look.
- Accent color: Muted teal (#468499) for actionable elements, such as buttons, links and headings.
- Body and headline font: 'Inter' (sans-serif) for a clean and readable experience. Inter is used because the prompt specified Inter.
- Use simple, outline-style icons to maintain a clean and trustworthy aesthetic.
- Minimalist layout with generous spacing and clear content separation to enhance readability.
- Subtle transitions and loading animations to provide a smooth user experience without being distracting.
